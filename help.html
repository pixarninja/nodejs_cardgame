Welcome to PlayMat!


Welcome to PlayMat, an exprimental card game developed with NodeJS and Docker!

This website is free to use. The website owner does not own any of the content on this website, nor do they take responsibility for information put on the website by its users. This website's source code (aside from sensitive backend data) is provided on GitHub. Amazon AWS is used to host the server and WebSocket connections.


How To

The game is implemented on Play Mat home page (port 80). WebSockets are instantiated as needed, for as many games as are going on. The chat message box you see to the right of the game page is tied to each game; players may communicate with each other there. If you would like to rename yourself in the chat server, you can do so by typing 'I am ' followed by your new name.

Players may join a game as an observer or player, or may start a new game if they wish. This dicision will be available from the Match Making link in the menu.
